#  Coffee Shop Website

A modern and responsive Coffee Shop website built using **HTML5** and **CSS3**. This project showcases semantic HTML, responsive layouts, CSS animations, and a coffee-themed user interface.

##  Preview

A simple coffee shop landing page featuring:

-  Home
-  Menu
-  About
-  Contact

##  Features

- Modern coffee-themed design
- Responsive layout
- Sticky navigation bar
- Smooth scrolling
- CSS animations
- Hover effects
- Card-based sections
- Beautiful typography using Google Fonts
- Mobile-friendly design

##  Technologies Used

- HTML5
- CSS3
